PROJECTS
Closet Collection

Closet Collection is your ultimate solution to organize your wardrobe. With this app, you can easily manage your clothing collection, take photos of your clothing items, create custom lists for trips or general organization, and categorize your wardrobe by color, season, and clothing type for easy filtering. You can add notes to your clothing items, such as if you have lent an item to a friend or if it is at the dry cleaners. In addition, you can create personalized lists for events or trips, and add items to the list as packing reminders. Finally, if you ever need to make changes, you can easily delete items from your closet or from a list. Say goodbye to cluttered closets and disorganized trips, and say hello to the perfect wardrobe management tool with Closet Collection.

FitLit

FitLit is a fitness tracking application that allows users to keep track of their sleep, hydration, and activity data. The application uses a local server and fetch requests to GET and POST data. Error handling has been added to both requests to catch any errors that may occur. The application features data visualization using charts from chart.js, presenting all data in a visually pleasing manner. Dayjs has been used to format and standardize dates. Users can add new data for the current day, and once submitted, the charts will update and populate with the latest data. Recursion has been utilized to construct a weeks worth of data if there are gaps in the data' date. A helper function file has been added to house functions that are used multiple times throughout the project, keeping the code DRY. The application also features a calendar that allows users to manipulate dates and view a wide variety of information from different dates. Users can view their personal information inside an information section.

Tech Used: Vanilla JS, Mocha, Chai, Git, GitHub, CSS, DayJS, ChartJS

PlayPal

PlayPal is an innovative mobile application designed to assist first-time mothers who are seeking ideas for safe, enjoyable, and stimulating activities for their children. This app provides a list of activities that can be filtered according to the location of the activity, whether it is an indoor or outdoor activity, and the gross and fine motor skills that the activity promotes.

PlayPalâ€™s UI is built using React and React Router with styling done using Module CSS. The backend service is created using Express and Postgres. We use Docker to create the Postgres database and write queries to the database using Knex. TablePlus is used to interact with the data in the database. The backend is deployed using Heroku and the frontend UI is deployed using Vercel.

Tech Used: React, React Router, selectJS Library, Cypress, Git, GitHub, Vercel, Postgres, ExpressJS, NodeJS, Docker

Cook Ahead

Coming up with meals ideas and making decisions for the family meal plans is hard. Cook Ahead is a meal planning application built with React and React Router. It allows users to generate a list of meal ideas and manage the meal plan by adding and deleting ideas. The application is tested using Cypress for end-to-end testing, and version control is managed with Git and GitHub. Postman is used to test API calls, and the application is deployed with Vercel.It features a responsive layout design with the use of media queries, ensuring optimal user experience across various devices.

Tech Used: React, React Router, HTML Tag Parser Library, Cypress, Git, GitHub, Vercel, CSS

News Reader

News Reader is an application that pulls articles from the New York Times API Top Stories API. This application allows users to stay up-to-date with the latest news and events by providing access to top stories from the New York Times. News Reader leverages various design patterns, including the Repository pattern, Container and Presentational Components pattern, and Hook pattern. The app is built using Create React App with Vite and is deployed on Vercel.

Tech Used: React, React Router, DayJS, Git, GitHub, Vercel, Postman , CSS
